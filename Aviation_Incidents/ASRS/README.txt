1. One dataset is from NASA <http://asrs.arc.nasa.gov> consisting of voluntarily reported safety knowledgebase from front-line aviation personnel. The data set is in csv format: 87 fields, 180,100 records. 

2. One dataset is from the FAA <http://wildlife.faa.gov> consisting of a wildlife strike database for civil aviation. It is also in csv format: 94 fields, 163,922 strike reports. 

Each line of these csv files is an incident with various information, such as phase of incident, weather condition.
